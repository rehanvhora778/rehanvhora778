## About Me

I build full-stack AI systems end to end — from RAG pipelines and LLM agents to the React dashboards that sit on top of them. My recent work spans retrieval-augmented chatbots, an AI email copilot with real Gmail integration, a YouTube-transcript RAG assistant with quiz generation, and a transfer-learning medical imaging classifier.

- 🧠 **AI Engineering** — RAG pipelines, LLM orchestration, and agentic, tool-calling workflows
- 🏗️ **Full Stack Delivery** — React/TypeScript on the front end, Flask/Django/FastAPI on the back end
- 🔐 **Security-Minded** — OAuth, JWT auth, row-level security, and human-in-the-loop safeguards on anything that sends or acts for a user
- 📈 **Consistent Output** — 2,300+ contributions in the past year across five production-style projects

<summary><b>🎬 yt-chat-ai — Chat with any YouTube video using GenAI</b></summary>
<br>

A full-stack RAG application that pulls a video's transcript (with an audio-transcription fallback for videos without captions), builds a FAISS vector index, and lets users ask questions, get summaries, and auto-generate quizzes from the content — in English or Hindi.

| | |
|---|---|
| **Problem Solved** | Long-form video content is slow to search or review manually |
| **Stack** | React · Flask · LangChain · Groq · Gemini · FAISS · MongoDB Atlas |
| **Core Feature** | RAG pipeline with timestamped chunk retrieval |
| **Auth & Security** | JWT-based registration and login |
| **Extras** | PDF/DOCX export, analytics dashboard, 5 selectable UI themes |
